# F1-2025-Season-Championship-Contenders-Project

This project analyzes the 2025 Formula 1 season for three drivers — Max Verstappen, Lando Norris, and Oscar Piastri using SQL for data preparation and Tableau for visualization. The goal is to compare qualifying performance, race results, racecraft, and head‑to‑head outcomes across the season.

Dataset
The analysis uses the Formula 1 dataset from Kaggle: https://www.kaggle.com/datasets/jtrotman/formula-1-race-data/data

Data was filtered to include only the 2025 season and the three selected drivers.

Project Overview
- The project answers key performance questions:
- How do the drivers compare in qualifying?
- How consistent are their race results?
- Who gains or loses the most positions on race day?
- How do they perform head‑to‑head in qualifying and races?
- What patterns appear across the season?
- SQL was used to clean, join, and aggregate the data. Tableau was used to build a structured, interactive dashboard.

Techniques Used
- SQL joins, grouping, filtering
- Data cleaning (handling \N, nulls, DNFs)
- Aggregation for finishing distributions
- Head‑to‑head comparisons

Key Metrics
- Average Qualifying Position
- Average Finishing Position
- Racecraft (positions gained/lost)
- Finishing Position Distribution
- Qualifying Head‑to‑Head
- Race Head‑to‑Head

Tools: SQL Server, Tableau
